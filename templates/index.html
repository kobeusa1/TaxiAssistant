About Our Porject

Data We Used

Raw NYC Taxi Trip Data that has been available to public.

Data Cleaning

We had around 4.1 million records on HDFS and remove dirty records (e.g. nulls, invalid geo coordinates, etc)

Feature Extraction: Used Geohashing to discretize the geo data which helps us to encode GPS point

Machine Learning Model

Random Forest Regression

Linear Regression

Findings and Next Steps
